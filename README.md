# Description
I have created a future ready Talent project which is Blog-Store. In my website I have used the languages HTML, CSS and JavaScript. In this website I have given 3 major sections Home, About and Contact section. This website is all about food blog. Here you can find all the tips and tricks and also people can find the different recipes to cook better and tasty food like cake, pastry etc. I have deployed my website with Microsoft Azure technology which is Static Web App. Firstly I claimed my student developer pack on github then I created a account on MS Azure then I have deployed this website.
